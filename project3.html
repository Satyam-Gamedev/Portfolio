Ring Master

Role:- Unity Game Developer

Tools:- Unity 2022, Visual Studio

Language:- C#

Overview

Ring Master is a simple but really fun toss game inspired by those fairground ring-throw stalls. I created a system where you can aim, throw the ring, and try to land it exactly on the objects placed on the table. Some objects are small, some are big. That made the game feel more skill-based instead of random.
